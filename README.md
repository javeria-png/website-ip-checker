

# 🌐 Website IP Address Checker - Python

This is my **second Python project!** A simple tool that finds the IP address of any website.

## Features
- 🔍 Find IP address of any website
- ✅ Works with or without https/www
- ❌ Shows error if website is invalid
- 🔁 Check multiple websites in one run

## How to Run

python ip_checker.py

## Example

Website: google.com
IP Address: 142.250.180.46

Website: github.com
IP Address: 140.82.121.4

## Requirements
- Python 3.x
- No extra libraries needed
